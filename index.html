Tic-Tac-Toe (Player vs BOT)

Select level to start

Game Rules

• Player who play first will be 'X' marker

• Player who play second will be 'O' marker

• Each player will take turn to start first

• Player will start the first game, & BOT will start the second game & vice versa

Notes

• 'Why you always win?' is base on random move on empty cell available

• 'Try hard not to lose' is base on random move but will block opponent winning move when possible

• 'Can you win me?' should not able to win a single game if logic is correct
